# RAGA Companion Web Prototype

Interactive prototype for thekarnatik.com: Karnatik and Hindustani raga practice, ear training, quiz, tanpura, metronome, concerts, and chord analysis tools.

## Run Locally

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Current MVP Areas

- Practice console with raga library
- Karnatik 72 Melakarta reference
- Raga theory quiz
- Ear training level 1
- Tanpura and pitch-aware metronome
- Concert finder beta page
- Chord Analyser with Karnatik/Hindustani notation views
